# GpChat
uniapp开发的纯nvue的即时聊天通讯App。
</br>
后台采用.net6,一套解决方案，分布式部署。
</br>
App采用uniapp的纯nvue，性能更好。
</br>
后台管理采用vue+elementui。
</br>
